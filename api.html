  • 거래소 거래 시세 정보

    https://oapi.planbit.io/public.ticker.dp/proc.go?currency={currency}

*{currency} = BTC, ETH, BCH, XRP

ex) https://oapi.planbit.io/public.ticker.dp/proc.go?currency=ETH
								//Sample Response
								
								{
									"result_code":"000",
									"data":{
										"volume":"721.00215101",
										"max_price":"698.261", 
										"close_price":"692.864",
										"min_price":"651.332",
										"sell_price":"692.864",
										"timestamp":"1525742630342",
										"currency":"ETH",
										"avg_price":"661.332",
										"buy_price":"690",
										"open_price":"672.112"
									}
								}
Key Description
result_code 결과 상태값 (정상 : 000)
open_price 최근 24시 내 시작 거래금액 (시가)
close_price 최근 24시 내 마지막 거래금액 (종가)
min_price 최근 24시 내 최저 거래금액 (저가)
max_price 최근 24시 내 최고 거래금액 (고가)
sell_price 현재 미체결 최저 판매가
buy_price 현재 미체결 최고 구매가
avg_price 최근 24시 내 평균 거래금액
volume 최근 24시 내 currency 거래량
currency Currency Symbol
timestamp 현재 시간

    https://oapi.planbit.io/public.order.dp/proc.go?currency={currency}

*{currency} = BTC, ETH, BCH, XRP

ex) https://oapi.planbit.io/public.order.dp/proc.go?currency=ETH
								//Sample Response
								
								{
									"result_code":"000",
									"data":{
									"timestamp":1525828419342,
									"order_currency":"ETH",
									"payment_currency":"USD",
									"list":[{
											"bids_quantity":"19.85",
											"asks_quantity":"27.89479323",
											"asks_price":"692.864",
											"bids_price":"690"
										}, 
										{
											"bids_quantity":"7.3262",
											"asks_quantity":"5.113548",
											"asks_price":"693",
											"bids_price":"689.136"
										}, 
											...
										],
									}
								}
Key Description
result_code 결과 상태값 (정상 : 000)
order_currency 결주문 코인 단위
payment_currency 결제 코인 단위
bids_quantity 구매 수량
asks_quantity 판매 수량
asks_price 판매 거래금액
bids_price 구매 거래금액
timestamp 현재 시간
